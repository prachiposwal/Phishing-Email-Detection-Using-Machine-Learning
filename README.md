# 🛡️ AI Powered Phishing Email Detection System

An end-to-end Machine Learning and Natural Language Processing (NLP) application for detecting phishing emails.

---

## 📌 Overview

Phishing emails are one of the most common cyber threats.

This project uses Machine Learning and Natural Language Processing to automatically classify emails as:

- ✅ Legitimate
- 🚨 Phishing

The application includes a professional Streamlit dashboard for prediction, model evaluation, and project insights.

---

# 🚀 Features

- 📧 AI Email Detection
- 🧠 NLP Text Preprocessing
- 📊 Performance Dashboard
- 📈 Project Insights
- 🔍 Suspicious Keyword Detection
- ⚠️ Risk Assessment
- 🎯 Confidence Score

# 🧠 Machine Learning Models

The following models were trained and compared:

- Linear SVM
- Logistic Regression
- Naive Bayes
- Random Forest
- Simple Neural Network

Final deployed model:

✅ **Linear SVM**

# 📊 Performance

| Model | Accuracy |
